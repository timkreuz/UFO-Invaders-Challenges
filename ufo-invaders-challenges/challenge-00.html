UFO Invaders: Setup


Introduction:

A few steps before we start coding... We'll need three things setup in order to get started: - Visual Studio Code - Git Bash - http-server

Goal #1: Install Visual Studio Code.

Visual Studio Code

Goal #2: Install Git and Setup the Code

Step 1 - Download Git Git Download Step 2 - Download the Code Create a folder called Git Right Click and select Git Bash Here Then run these two commands:
git clone https://github.com/timkreuz/UFO-Invaders.git git clone https://github.com/timkreuz/UFO-Invaders-Challenges.git
This should create two folders on your machine. One for the starter code. A second for the directions. Open the folder for the the starter code and then checkout the starter code.
cd UFO-Invaders/ git checkout cc00-starter
Step 3 - Load the Code in Visual Studio
File -> Open Folder - Select UFO Invaders folder from your Git folder. Save the Workspace.

Goal #3: Install http-server

Step 1 - Install Open a command prompt:
npm install http-server -g
Open your ufo invaders folder. Right Shift-Click to open the Power Shell. And start up the server.
http-server
Now let's open up the game. And bookmark this. UFO Invaders
