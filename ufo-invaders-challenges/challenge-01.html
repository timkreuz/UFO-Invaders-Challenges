Step 4 - Now let's randomize the ufo. All we need to do is change the speeds and set a new timer. This might look weird, but it's just 3 lines of code. // Speeds will be between -4.5 and 4.5 ufo.speedX = (Math.floor(Math.random()* 90) - 45)/10; ufo.speedY = (Math.floor(Math.random()* 90) - 45)/10; // Timer will be between 2 and 3 seconds in the future. ufo.timer = Date.now() + Math.floor(Math.random() * 1000) + 2000;
